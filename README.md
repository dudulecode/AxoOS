# AxoOS

**AxoOS** is a fake operating system entirely written in Python. **AxoOS** is **ONLY** available on Windows 10/11, no linux or MacOS version is scheduled for now.

## Why?

I've made two fake operating system already, [MaxPyOS](https://github.com/miyucode/MaxPyOS) and [PandoOS](https://github.com/miyucode/PandoOS), this was 4 years ago, i didn't code in years but I decided to code again for fun, so here's AxoOS, my new Python project.
